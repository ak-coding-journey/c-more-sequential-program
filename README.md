# C More Sequential Program

This repository contains a single C program that runs **three tasks sequentially**:

1. **Profit or Loss Calculation**  
   - Input: cost price and selling price  
   - Output: profit, loss, or no profit/loss  

2. **Leap Year Check**  
   - Input: a year  
   - Output: whether it is a leap year  

3. **Divisibility by 5 Check**  
   - Input: an integer  
   - Output: whether it is divisible by 5
